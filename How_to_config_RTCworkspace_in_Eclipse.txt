1) In Eclipse, File -> OpenProjects from File System : select the RTCworkspace folder, import the project "AdminOCProj".
2) Properties for AdminOCProj project-> Java Build Path-> Add External JARs: select all RFT jars to add.
3) After AdminOCProj project refreshed, there is still a error: "The import com.ibm.jsse2 cannot be resolved" caused by ADB0NAEA.java. 
   Need to add "ibmjsseprovider2.jar" into Java Build Path, which is in "C:\Program Files (x86)\IBM\SDP\jdk\jre\lib" folder.(or "C:\Program Files (x86)\
   IBM\Java70\jre\lib" folder)
