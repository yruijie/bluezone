1) In Eclipse, File -> OpenProjects from File System : select the RTCworkspace folder, import the project "AdminOCProj".
2) Properties for AdminOCProj project-> Java Build Path-> Add External JARs: select all RFT jars to add.
3) After AdminOCProj project refreshed, there is still a error: "The import com.ibm.jsse2 cannot be resolved" caused by 
   ADB0NAEA.java. 
   Need to add "ibmjsseprovider2.jar" into Java Build Path, which is in "C:\Program Files (x86)\IBM\SDP\jdk\jre\lib" folder.
   (or "C:\Program Files (x86)\IBM\Java70\jre\lib" folder)
4) How to run these test case:
   a. Eclipse安装testNG plug-in.
   b. Test case中，添加：
      import org.testng.annotations.*;
      import org.testng.asserts.LoggingAssert;
      import org.testng.Reporter;
      在类声明之前添加@Test。即public class XYZ extends XYZHelper的前一行。
       
   c. Run Configurations -> TsetNG, 新建一个configuration, Project为“AdminOCProj”， Class为“包名.类名”。
   
   d. 因为没有安装FTE插件，所以运行不起来
